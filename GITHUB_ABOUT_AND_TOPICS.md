# GitHub Repository Settings / GitHub 仓库设置建议

> [中文] 这个文件不用给用户阅读，主要用于设置 GitHub 右侧 About 和 Topics。  
> [English] This file is mainly a checklist for the GitHub sidebar About section and repository Topics.

## About 描述 / About Description

**[中文]**  
如果 GitHub 右侧 About 只能写一段，建议使用英文，因为它更适合 OpenAI 开源项目审核和国际访问者。

**[English]**  
If the GitHub About field only has room for one paragraph, use English because it is better for OpenAI open-source review and international visitors.

```text
Free, open-source, local-first scheduling assistant for hemodialysis rooms. Privacy-friendly dialysis scheduling with patient/staff/machine management, infection zones, fixed seats, and safety checks.
```

**[中文备用]**

```text
面向血液透析室的免费开源本地排班辅助工具，支持患者、医护、机器、传染分区、固定机位和排班安全自检。
```

## GitHub Pages / 在线演示

**[中文]**  
如果启用 GitHub Pages，请只使用假数据演示，不要公开真实患者资料。

**[English]**  
If GitHub Pages is enabled, use fake demo data only. Do not publish real patient data.

**[中文步骤]**

1. 打开仓库 Settings。
2. 进入 Pages。
3. Branch 选择 `main`。
4. Folder 选择 `/root`。
5. 保存后打开生成的网址，确认页面可以加载。

**[English Steps]**

1. Open repository Settings.
2. Go to Pages.
3. Select branch: `main`.
4. Select folder: `/root`.
5. Save and open the generated URL to verify that the app loads.
